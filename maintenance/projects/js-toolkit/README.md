## Setup

1. Install NodeJS >= [v10.15.1](http://nodejs.org/dist/v10.15.1/), if you don't have it yet.

2. Run the bootstrap script to install local dependencies and link packages together:

```sh
yarn install
```

3. Build all packages

```sh
yarn build
```

4. Run tests:

```sh
yarn test
```
